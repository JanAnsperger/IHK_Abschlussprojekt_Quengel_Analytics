Quengel_Analytics – Analyse & Modellierung zur Vorhersage von Kinderhaushalten

Dieses Repository enthält alle Komponenten meines IHK-Abschlussprojekts zur Analyse und Vorhersage, ob in einem Haushalt Kinder vorhanden sind, basierend auf einem Jahr Supermarktdaten.
Das Projekt wurde für das fiktive Unternehmen Quengel-Analytics umgesetzt.

Projektziel:

Ziel ist es, anhand von Kundendaten ein Modell zu entwickeln, das zuverlässig vorhersagen kann:

„Gibt es im Haushalt Kinder?“

Gleichzeitig soll ein Personenprofil erstellt werden, das beschreibt, welche Merkmale einen Kunden mit mindestens einem Kind auszeichnen.
Die Analyse soll Mitarbeitenden über ein Power-BI-Dashboard zugänglich gemacht werden.
